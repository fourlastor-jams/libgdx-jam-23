## Keys:

### General

- Mute music: `M`
- Start fight: `Space`

### Player 1

- Left: `A`
- Right: `D`
- Attack: `SPACE`

### Player 2

- Left: `LEFT ARROW`
- Right: `RIGHT ARROW`
- Attack: `ENTER`
